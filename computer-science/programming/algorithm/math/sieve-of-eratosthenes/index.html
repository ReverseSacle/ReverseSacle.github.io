# Sieve of Eratosthenes

# 实质

埃拉托斯特尼筛法(wiki)

该算法主要是用于素数，根据素数的定义只有1和它本身两个因数的自然数，叫质数或称素数（就是除了1和它本身外，不能再被其他数整除），埃拉托斯特尼筛法通过倍数关系，将由素数倍数得到的数排除，以此来筛选出素数。

# 核心

  • 素数的倍数关系

由倍数得到的数除了1和它本身外，还有倍数基础的素数，为此由倍数得到的数不为素数。

# 优化

通过观察由埃拉托斯特尼筛法得到的素数集合可知，除了2以外，素数不会再出现在偶数上，为此当求取素数时通过直接排除偶数来减少计算量。

假设素数有n2\frac{n}{2}2n​个

由于排除的时候也要使倍数乘积之后的数保持奇数，当以倍数关系为基础时，奇数的偶数倍数会导向偶数，为此倍数乘积时也需保持奇数倍数。

每当得到一个奇数倍数值时，减少一个素数，以此往复，最后剩下的奇数(除了1，包括2)便是素数了。
