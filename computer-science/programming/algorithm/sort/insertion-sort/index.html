# 直接插入排序(Straight Insertion Sort)

是一种比较直觉的排序方法，该方法每次将当前值插入到之前已排序数组的合适位置。

其实现方法较为简单，设nin_ini​，ni−jn_{i-j}ni−j​，往回比较至nin_ini​的值不小于ni−jn_{i-j}ni−j​的值为止，比较途中将已比较的值后移一位，最后再将nin_ini​的值插入到比较停止的位置。

Straight Insertion Sort

时间复杂度：O(n2n^2n2)

空间复杂度：O(1)

 

# 折半插入排序(Binary Insertion Sort)

基于直接插入排序的优化排序方法。

由于直接插入排序中往回查看中有大量的数值比较次数，其通过二分查找预先确定了当前值在之前已排序数组中的最适位置，以此减少了往回查找的比较次数。

其实现方法较为简单，设nin_ini​，njn_jnj​，先通过二分查找确定njn_jnj​，再将[njn_jnj​,ni−1n_{i - 1}ni−1​]范围内的所有值后移一位，最后再将nin_ini​的值插入到njn_jnj​中。

Binary Insertion Sort

时间复杂度：O(n2n^2n2)

空间复杂度：O(1)

 

# 二路插入排序(Two-Way Insertion Sort)

基于直接插入排序的优化排序方法。

由于直接插入排序中往回查看比较中有大量的数值交换移动次数，其通过head游标与tail游标来辅助值的插入，以此减少了往回查找中的数值交换移动次数。

其实现方法较为简单，设nin_ini​，nheadn_{head}nhead​，ntailn_{tail}ntail​，倘若nin_ini​值小于nheadn_{head}nhead​值就在nheadn_{head}nhead​的前一位插入；倘若nin_ini​值大于ntailn_{tail}ntail​值就在ntailn_{tail}ntail​的后一位插入；当nin_ini​值既不小于nheadn_{head}nhead​值也不大于ntailn_{tail}ntail​值，就从ntailn_{tail}ntail​往前比较至nin_ini​值不小于当前位置值为止，比较途中将已比较的值后移一位，再将nin_ini​值插入到比较停止的位置。

Two-Way Insertion Sort

时间复杂度：O(n2n^2n2)

空间复杂度：O(nnn)

 

# 表插入排序(Table Insertion Sort)

基于直接插入排序的优化方法。

有点类似二路插入排序，操作实现原理跟其基本一致，也是减少了往回查找的数值交换移动次数。

其实现方法有点复杂，但实现原理比较简单，简直就是二路插入排序的链表版本。创建以数组形式模拟链表的静态链表，将其首元素当作虚拟头，设nin_ini​，nheadn_{head}nhead​，ntailn_{tail}ntail​，倘若nin_ini​值小于nheadn_{head}nhead​值就在nheadn_{head}nhead​的前一位插入；倘若nin_ini​值大于ntailn_{tail}ntail​值就在ntailn_{tail}ntail​的后一位插入；当nin_ini​值既不小于nheadn_{head}nhead​值也不大于ntailn_{tail}ntail​值，就从nheadn_{head}nhead​往后比较至nin_ini​值不小于所在位置的值为止，再将nin_ini​值插入到比较停止的位置。

Table Insertion Sort

时间复杂度：O(n2n^2n2)

空间复杂度：O(nnn)

 

# ! 希尔插入排序(Shell Insertion Sort)

基于直接插入排序的优化方法，又称递减增量排序/缩小增量排序(Diminishing Increment Sort)。

该算法基于思想：若待排记录序列为正序时，其时间复杂度可提高至O(n)。该算法在n值很小时效率较高，由于增量数组的选取仍是一个尚未解决的难题，故没有一种最好的增量数组。

希尔排序是通过将原始数组分为若干个子序列，每个子序列分别进行插入排序，然后逐步缩小子序列的间隔，最终完成排序。

主要方法是通过比较和交换跨越较大间隔的元素，将较小的元素逐渐移到数组的前部。这样可以使得数组变得部分有序，接下来再进行插入排序时，每次移动的元素较少，从而提高整体排序的效率。

其实现方法较为简单，创建一个数量为mmm的增量数组，按照增量数组内的增量值δ\deltaδ依次对原数组进行总mmm次的增量值δ\deltaδ插入排序。

Shell Insertion Sort

时间复杂度：O(nlognnlognnlogn) ~ O(n2n^2n2)

空间复杂度：O(1)
