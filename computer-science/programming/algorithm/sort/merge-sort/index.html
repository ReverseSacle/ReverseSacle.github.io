# ! 二路归并排序(Two-Way Merge Sort)

二路归并排序基于归并的思想实现的排序方法，其归并的含义是将两个或两个以上的有序表组合成一个新的有序表。由于其基础形式是通过递归所实现，其实用性较差，但与其它排序方法相比，其最大特点是稳定。

其实现方法较为简单，有点类似快速交换排序，以数组中间为分界线，划分左右区域，接着再在左右区域中以中间分界线接着划分左右区域，以此往复，最后在将左右区域合并返回。通过双指针的方法实现合并操作，设pi(i=left,i<=pmid), pj(j=mid+1,j<=pright)\small p_i{(i = left,i <= p_{mid})}\text{, }p_j{(j = mid + 1,j <= p_{right})}pi​(i=left,i<=pmid​), pj​(j=mid+1,j<=pright​)，将[pleft,pright\small p_{left}, p_{right}pleft​,pright​ ]区间内的值存入到辅助空间内，再在原数组[left,right\small left,rightleft,right ]区间内依次选取pi\small p_ipi​与pj\small p_jpj​两者中的最小值。

Two-Way Merge Sort(Recursive)

时间复杂度：O(nlognnlog nnlogn)

空间复杂度：O(n+mn+mn+m)

 

由于依靠递归实现的二路归并排序使用了除其辅助排序空间外的额外空间，这使其降低了实用性。但其算法中有明显的规律模式，因此可以通过循环代替递归。其规律模式：假设数组长度为nnn，每次迭代便会得到⌈n2⌉\small \lceil \frac{n}{2} \rceil⌈2n​⌉的左右子数组，子数组两两归并并返回。

Two-Way Merge Sort(Non-Recursive)

时间复杂度：O(nlognnlog nnlogn)

空间复杂度：O(nnn)
