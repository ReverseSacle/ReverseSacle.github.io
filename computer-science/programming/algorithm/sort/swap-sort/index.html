# 冒泡交换排序(Bubble Swap Sort)

由于该算法的过程就像：水中气泡(数值较小值)逐趟向上漂浮，而石块(数值较大的值)往下沉，每一趟都有一块最重的石头沉入到水底，故称冒泡(起泡)排序。

其实现原理较为简单，设nin_ini​，ni+1n_{i+1}ni+1​，排序中，比较nin_ini​与ni+1n_{i+1}ni+1​的值，当nin_ini​值大于ni+1n_{i+1}ni+1​值，便会交换nin_ini​与ni+1n_{i+1}ni+1​的值，每一次排序都会将(n−in-in−i)数组中的最大值放置最后，每完成一趟排序都会减少一个需要排序的值。

Bubble Swap Sort

时间复杂度：O(n2n^2n2)

空间复杂度：O(1)

 

# ! 快速交换排序(Quick Swap Sort)

快速交换排序采用了分治的策略，通过递归地将数组分成较小的子数组来实现排序。实现过程中是先选择一个基准元素，将数组中小于基准的元素放在基准的左边，大于基准的元素放在右边，然后对左右两个子数组分别递归地进行同样的操作。

其实现方法有点复杂，但实现原理较为简单，把首元素当作枢纽，先在枢纽右边找小于或等于枢纽的值，找到后该值与枢纽值交换位置，再在枢纽左边找大于或等于枢纽的值，找到后该值与枢纽值交换位置。每完成一趟排序都会使枢纽其左边全部值都小于或等于枢纽值，其右边全部值都大于或等于枢纽值，确定的枢纽会将该数组划分为两个子区域，接着对两个子区域进行排序，以此往复最终完成排序。

Quick Swap Sort

时间复杂度：O(nlognnlognnlogn) ~ O(n2n^2n2)

空间复杂度：O(nnn)
