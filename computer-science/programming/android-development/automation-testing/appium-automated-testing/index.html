# Desired Capabilities常用关键字

Desired Capabilities关键字在启动时会传递给Appium Server。

Key值描述
platformName'ISO' 或 'Android' 或 'FirefoxOS'手机所属操作系统
platformVersion'14'手机所属操作系统的版本，例如安卓14中的
automationName自动化测试的引擎
deviceName'Android' 或 'Android Emulator'充当手机设备的类型，安卓手机可随意填写
noResetTrue 或 False执行完脚本后是否完全清空app数据，False表示同意清空
unicodeKeyboardTrue 或 False是否使用自带输入法输入中文，True表示同意
resetKeyboardTrue 或 False执行完输入法输入程序后是否恢复成原来的输入法，True表示同意
language为模拟器设置语言
locale为模拟器设置所在区域
appPackageAndroid 独有，Android应用的包名
appActivityAndroid 独有，Package中所要启动的 Android acticity

可阅览资料：→Appium Python Driver Api←

# 安全阻拦解决方法

报错java.lang.SecurityException: Permission Denial:

有两种可能，一是appActivity并不是一开启动的，二是代码防止需通过反编译。

反编译后打开AndroidManifest.xml再<activity>标签中找到appActivity并在其标签中添加android:exported="true"，再编译加签名。
