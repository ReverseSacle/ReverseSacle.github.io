# 思考过程解析

创建Pascal's Triangle的基本思路是模拟动图的实现，每一行中，除第一个值和最后一个值外，每一个值都是由前一行前一列的值加上前一行当前列的值的和。

——————————————

当想获取特定行的每一个值时，最基本的思路是模仿上面的实现过程直到实现完特定行为止。

倘若此时模仿下基本思路的求值过程，并将其稍微细化

由上面过程可知，

当有一行时，默认一个元素值为1，

当超过一行时，每一行计算时的过程相当于将整体后移一位并加上其原本所在位置的值。

——————————————

此时已无法再从上面得到什么信息了，接下来直接看Pascal's Triangle的定义。
