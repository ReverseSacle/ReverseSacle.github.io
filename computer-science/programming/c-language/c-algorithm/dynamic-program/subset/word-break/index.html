# 思考过程解析

例如

s = "catsanddog"

wordDict = ["and,cat,cats,dog"]

最基本的思路是字符串的子串一个个拟合，如下

c => F ca => F cat => T

s => F sa => F san => F sand => F

sandd => F sanddog => F

cats => T

a => F an => F and => T

d => F do => F dog => T

catsa => F catsan => F catsand => F

catsandd => F catsanddo => F catsanddog => F

实际上，当cats、and与dog拟合后就会返回true了。
