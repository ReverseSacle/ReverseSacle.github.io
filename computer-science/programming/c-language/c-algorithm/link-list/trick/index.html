LC.21 Merge Two Sorted Lists

通过双指针指向当前最小值的结点，再由该双指针使之前最小值的结点指向其下一个结点。也可以将双指针用于一个新的链表结点空间，每次选取完最小值后，再在该双指针所存储的空间内存储最小值结点的内存，最后再由该双指针指向其下一位空间，以此用于下一次的存储。

由于单向链表的方向是单一的，但可通过指针指向其下一个结点。
