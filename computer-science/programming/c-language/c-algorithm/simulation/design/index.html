LC.895 Maximum Frequency Stack

Approach1

FreqStack struct

由unordered_map<int,int>、unordered_map<int,ListNode>与maxfreq整型变量组成。

通过unordered_map<int,int>记录每个push值与其当前出现次数；再创建LinkList结构，通过unordered_map<int,ListNode>记录出现的次数与对应次数下有哪些值；通过maxfreq整型变量记录当前最大出现次数。

Approach2

FreqStack struct

由unordered_map<int,int>、priority_queue<pair<int,pair<int,int>>>与pos整型变量组成。

通过unordered_map<int,int>记录每个push值与其当前出现次数；通过priority_queue<pair<int,pair<int,int>>>记录{出现次数,{第几个被添加,值}}；通过pos整型变量记录当前添加值为第几个被添加。

priority_queue内会先比较出现次数，出现次数高的值会在顶部；当出现次数一样时，会比较第几个被添加，以此模拟栈的后进先出。
