LC.168 Excel Sheet Column Title

Excel Sheet Column Title

Excel 采用了26进制(26-nary)进位，但不包括0。而实际的26进制系统中A应该从0开始计数，而Excel中的A是从1开始计数，因此从字母换算成数值时需加一以填补Excel系统与实际的26进制系统空缺；从数值换算成字符时需减一。

比如通常的10进制进位，

 754=7∗102+5∗101+4∗100\small \ 754 = 7*10^2 + 5*10^1 + 4*10^0 754=7∗102+5∗101+4∗100 。

而Excel采用的26进制，

 ABZ(754)=(A+1)∗262+(B+1)∗261+(Z+1)∗260\small \ ABZ(754) = (A + 1)*26^2 + (B + 1)*26^1 + (Z+1)*26^0 ABZ(754)=(A+1)∗262+(B+1)∗261+(Z+1)∗260。
