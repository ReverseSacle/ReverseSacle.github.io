# 队列(Queue)

队列是一种基本的数据结构，它按照先进先出(First In First Out - FIFO)的原则管理数据，类似于现实生活中排队的场景。队列支持在一端(尾部)添加元素，而在另一端(头部)移除元素。

基本特性

  • 先进先出 - 队列中先进入的元素将先被移除，后进入的元素将后被移除，保持了元素的顺序
  • 两个基本操作 - 入队(Enqueue)向队列尾部添加元素和出队(Dequeue)从队列头部移除元素
  • 不支持随机访问 - 只能在队尾进行插入，队头进行删除

# 序列队列

使用数组来存储队列中的元素，通过两个指针分别指向队列的头部和尾部，入队和出队的时间复杂度为O(1)。

# 链队列

使用链表来实现队列，每个节点包含数据和指向下一个节点的指针，可以更灵活地处理动态大小的队列。

# 循环序列队列

对于序列队列来说，序列队列与序列栈的基本原则恰恰相反。但，对于循环的顺序队列来说，应满足当存储空间满时还可添加的原则，因此，倘若想实现循环队列的基本功能，一是通过设置信标来判断空间是否满；二是通过减少一位存储空间。

# 双端队列(Double-Ended Queue - Deque)

双端队列是一种允许在两端进行插入和删除操作的线性数据结构。它可以在队列的两端同时进行入队(enqueue)和出队(dequeue)操作，具有先进先出(FIFO)和后进先出(LIFO)的特性。双端队列可以在前端和后端进行快速的插入和删除操作，因此在一些场景中比普通队列更灵活。
