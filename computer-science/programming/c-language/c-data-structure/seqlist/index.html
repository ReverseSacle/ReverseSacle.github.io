# 顺序表(Sequential List)

顺序表是一种基本的线性表数据结构，它采用一组连续的地址存储单元来依次存储数据元素，元素之间的顺序关系由它们在内存中的相对位置表示。

顺序表可以是静态的(由静态数组实现)或动态的(由动态数组或链表实现)。

基本特性

  • 连续存储空间 - 元素在内存中是连续存储的
  • 固定大小 - 静态顺序表的大小在创建时就确定，而动态顺序表的大小可以动态调整
  • 随机访问 - 通过下标可以直接访问任意位置的元素，时间复杂度为O(1)

当构建一个线性结构时应满足以下条件

  • 唯一的首元素与唯一的尾元素
  • 除第一个元素外，其余每个元素只有一个头元素
  • 除最后一个元素外，其余每个元素只有一个尾元素
