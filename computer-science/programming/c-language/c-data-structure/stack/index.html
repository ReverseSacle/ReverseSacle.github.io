# 栈(Stack)

栈是一种基本的数据结构，它按照后进先出(Last In First Out - LIFO)的原则进行操作。栈可以看作是一系列元素的集合，其中插入和删除元素的操作仅在栈的一端进行，通常称为栈顶。栈的基本操作包括压入(push)元素到栈顶和弹出(pop)栈顶元素。

基本特性

  • 后进先出(LIFO) - 最后入栈的元素是第一个被弹出的
  • 只在一端进行操作 - 元素的插入和删除只在栈顶进行
