# 概述

Linux是一个自由、开放源码的类似Unix的操作系统。严格来讲，Linux这个词本身只表示Linux内核，单独的Linux内核并不是一个可以正常工作的操作系统，为此，就有了各种Linux发行版。内核主要负责系统的内存管理，硬件设备的管理，文件系统的管理以及应用程序的管理。

Linux之父(林纳斯·本纳第克特·托瓦兹 - Linus Benedict Torvalds)发起了这个开源项目，是Linux内核的最早作者，担任Linux内核的首要架构师，此外还发起了Git这个开源项目，并为主要的开发者。

  • 1989年 - Linus Torvalds进入芬兰陆军新地区旅，服11个月的国家义务兵役，军衔为少尉，主要服务于计算机部门，任务是弹道计算。服役期间，购买了安德鲁·斯图尔特·塔能鲍姆所著的教科书及minix源代码，开始研究操作系统

  • 1990年 - Linus Torvalds退伍后回到大学，开始接触 Unix。Minix是一个迷你版本的类 Unix 操作系统，由塔能鲍姆教授为了教学之用而创作，采用微核心设计

  • 1991年 - Linus Torvalds开源了Linux内核

Linus Torvalds开源的只是Linux内核，一些组织或厂商将Linux内核与各种软件和文档包装起来，并提供系统安装界面和系统配置、设定与管理工具，这就构成了Linux 的发行版本。Linux 的发行版本可以大体分为两类

  • 商业公司维护的发行版本 - 比如Red Hat公司维护支持的Red Hat Enterprise Linux(RHEL)
  • 社区组织维护的发行版本 - 比如基于Red Hat Enterprise Linux(RHEL)的CentOS，基于Debian的Ubuntu

# 安装过程

VMware虚拟机安装前置完成后，重启后再打开虚拟机，点击创建新的虚拟机，基本上都是点下一步，除了

  • 命名虚拟机页面，名称和位置可自行选择
  • 指定磁盘容量页面，选将虚拟磁盘存储为单个文件
  • 指定磁盘文件页面，磁盘文件位置可自行选择

创建完成后，自动打开，看见字体后按下回车键，出现图像化界面后语言选择中文→简中。

图形化界面更新后，滑到下方点击有感叹号的地方(安装位置)，进入后点击左上角的完成按钮。主界面点击开始安装，等待进度条走完，期间，点击Root密码，设置后点完成配置。等待进度条走完，点击右下角的重启，重启后输入root，接着输入之前设置的密码，输入vi /etc/sysconfig/network-scripts/ifcfg-ens33指令(如果有看到ATTENTION的提示就按回车键)，之后按键盘I键，查看有没有ONBOOT=no的参数记录，如果有就通过键盘方向键移至其位置将no改成yes，没有就直接在里面添加ONBOOT=yes，修改完成后按键盘Esc，输入:x再回车即可保存并退出，之后将鼠标移至虚拟机页面上方的橙色等于号位置，点击旁边的向下箭头，选择关闭客户机。

# 通过SecureCRT客户端连接服务器

SecureCRT-8.7.2-注册解除版(密码：eais)

打开Vmare虚拟机开启CentOS7虚拟环境，进入后输入root，再输入密码，之后再输入ip addr命令。

下载完成后，解压，双击SecureCRTPortable.exe运行程序，界面出现后有个小界面，鼠标移至会话管理 → Sessions → 新建会话 → 弹出一个新建会话向导界面，打勾不使用这个向导，点击完成，接着弹出会话选项-新建界面。

返回虚拟机，看到link/ether的下一行，记住inet后面的第一个ip地址，/24不需要，返回到SecureCRT，名称处输入ip地址，接着点击左边的SSH2，主机名输入ip地址(云服务器就输入公网ip)，接着点击左边的终端，点击右边的发送协议并将其改为30秒，左边点击外观，右边的当前颜色方案选传统(按喜好改)，再点右边的字体按钮，字体选为新宋体，其他自行选择，确定后，编码选择UTF-8，修改完成后点下边的确定，再点连接，界面更新后，出现小窗口，点接受并保存，接着在口令处输入Linux(CentOS7)的密码。
