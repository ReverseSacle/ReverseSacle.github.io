# Debug mode

开启Debug模式后可以实时代码同步浏览器内容更新，在浏览器上能看到错误信息。

Pycharm → Run图标左边的下拉按钮 → Edit Configurations.... → Modify options → Flask debug → Apply

# ORM(对象关系映射 - Object-Relationship-Mapping)模型

一种可以用Python面向对象的方式来操作关系型数据库的技术，具有可以映射到数据库表能力的Python类，称为ORM模型。

优点 - 开发效率高，安全性强，灵活性强不同数据库可直接替换
