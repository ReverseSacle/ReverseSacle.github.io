channels用于查找和安装软件包，这里相关部分为其添加了环境包下载源 - 清华镜像源

# conda相关命令

conda create -n env_name python=3.9 - 创建独立环境，env_name为自定义环境名，python=3.9即指定了3.9版本的python环境

conda env list - 查看当前有哪些环境

conda remove -n env_name --all -y - 移除独立环境，env_name为自定义环境名

conda activate env_name - cmd或Anaconda Prompt中切换到独立环境

conda install package_name - 安装环境包，先用conda activate env_name切换到指定独立环境，然后再使用该命令安装环境包

pip install package_name - conda install命令无法下载时，使用该命令

conda install --rev 0 - 重置conda当前环境到最初始的状态

conda (command) -c https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/ - 临时换源

conda config --show channels - 查看所有channels

conda config --remove-key channels - 换回conda的默认源

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/ - 添加指定channels

conda config --remove channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/ - 删除指定channels

# 使用conda

File → Settings... → Project: 你的项目文件名 → Python Interpreter，

或者点击右下角的Python → Interprete Settings...

进入之后点击右边顶栏的Python Interpreter → Show All... → 左边的加号 → Add Local Interpreter... → Conda Environment，点击右边的Interpreter即可看见刚刚通过conda create -n命令创建的独立环境。

# 关闭AI辅助

File → Settings... → 搜索plugin → 下滑到Local AI/ML Tools → Disable all → OK并重启

# 关闭粘贴代码时自动格式化

File → Settings... → 搜索smart keys → 将Reformat on paste选为None → 搜索code style → Python → Spaces → 将Around operators中不要的都关掉

# VS Code中使用conda

首先在EXTENSIONS中安装Microsoft的Python插件，使用Ctrl Shift P快捷键 → 输入Select Interpreter → 选择Python: Select Interpreter → 选择刚刚通过conda create -n命令创建的独立环境
