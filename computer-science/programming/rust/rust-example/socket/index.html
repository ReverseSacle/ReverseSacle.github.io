# 基本

std::net是用于TCP/UDP网络连接的标准库模块。

TcpListener与TcpStream是std包中net模块下的项(item)，用于TCP连接。

TcpListener用于服务器，

创建socket的关联函数签名为 - pub fn bind<A: ToSocketAddrs>(addr: A) -> Result<TcpListener>

TcpStream用于TCP流，

创建socket的关联函数签名为 - pub fn connect<A: ToSocketAddrs>(addr: A) -> Result<TcpStream>

两者的参数是一个具有ToSocketAddrs特征的泛型。返回值为一个具有Tcp结构体类型的Result枚举。
