# 常见防爬虫手段

服务端

  • 限制请求速率

  • 限制地域/ASN

  • 限制IP/域名

  • 访问验证/api链接加密

  • 事件转移 - 重定向，端口转移

  • 本地防火墙 - 自定义规则WAF防火墙

  • 云端防火墙 - Cloudflare

  • 限制所有访问途径，根据自定义规则来允许持有特定条件的对象的访问

客户端 - 前端网页

  • 慢加载/慢渲染(前后端分离) - 用js渲染html，访问链接时由于js未渲染导致信息不存在
  • 验证器 - 验证码/滑动验证/逻辑验证/js验证/真人图像识别/真人逻辑思维识别

客户端 - App

  • UI隐藏 - 某些App爬取一定时间后，UI突然找不到了，但是App上仍然能够看到

# 常见爬虫手段

  • 某个链接是目标，但无防御手段 - 直接下载该链接的内容
  • 某个网页标签内的文本是目标，但无防御手段 - Request/BeautifulSoup
  • 某个网页标签内的文本是目标，但有api链接 - js逆向与解密，之后直接通过该api获取信息
  • 某个网页标签内的文本是目标，并且不是慢渲染，无验证器 - DOM解析后直接文本提取
  • 某个网页标签内的文本是目标，并且是慢渲染，有/无验证器 - Selenium/Puppeteer/DrissionPage
  • 某个网页标签内的文本是目标，并且是慢渲染，有/无验证器，但dom无法被获取到，也没js限制复制 - Selenium类自动化工具 + 代码实现Ctrl + A, Ctrl + C, Ctrl + V
  • 某个网页标签内的文本是目标，是/不是慢渲染，有/无验证器 - Python + OCR
  • 某个App内的页面内容是目标，有api链接 - 抓包之后分析链接，js逆向与解密，之后直接通过该api获取信息
  • 某个App内的页面内容是目标，无客户端防御手段 - Appium
  • 某个App内的页面内容是目标，有客户端防御手段但无法解决 - Python + adb + OCR
