注意事项(推荐配置前先阅读)

关于头像位移异常的问题

刚开始搭建shoka主题时，推荐./themes/shoka/_config.yml文件中的statics参数设为/。倘若不是，当CDN加速链接缓冲未完成时会产生此问题。此外，强力推荐当未完全配置完主页功能时将statics参数配置为/，配置完后再改成CDN加速链接。

关于音乐播放功能

倘若你的音乐配置引用的是国外链接，在国内使用时，由于网络问题，可能会出现无法播放的情况(依据是你的国外音乐链接官网ip地址是否被墙)；倘若你的音乐配置引用的是国内的链接，在国外使用时也存在由于网络问题而导致无法播放的情况。

关于评论功能无效的问题

一是由于./themes/shoka/_config.yml文件里的valine设置参数没有正确配置好；二是由于LeanCloud不同版本对应的服务政策不同，比如：倘若你是用LeanCloud国际版，在国内使用时会出现评论区消失的问题；倘若你是用LeanCloud华北\华东版，在国外使用时，存在会出现评论区消失的问题(与自身网络问题有关)。

Hexo文章常用命令

  • hexo new post 标题与文件名 → 发布一篇新文章
  • hexo new page 标题与文件夹名 → 发布一个新的页面(例如顶栏的菜单按钮对应的页面)

文章顶栏Front Matter配置

  • tags(标签) → 会将文章添加到指定标签目录中

  • cover(封面) → 指定图片覆盖当前文章的封面

  • sticky(置顶) → 手动设置是否将该文章添加到置顶区域

  • type(类型) → 配置新页面而非文章时需添加指定类型

  • categories(目录分层) → 手动指定当前文章的目录索引路径，用于分类各文章

  • 关于categories，倘若网站语言配置为简体，文件夹路径名称都为英文，但想要使部署后的精选翻转卡片和网络链接地址等地方显示中文，则需配置对应的中文映射(此时文章中的categories参数仍填入中文形式)

搜索功能的详细介绍
Algolia官网

前往Algolia官网主页

  • 官网完成注册后，进入welcome页面→点击API keys，在All API Keys页面处点击New API key并填写相关信息

  • 新建好API key之后，在hexo根目录的_config.yml文件中配置algolia相关参数(可在Algolia官网页面→API Keys页面→(Your API Keys | All API Keys)页面中查看)

  • 配置好后，在hexo根目录右键点击Git Bash Here，输入hexo clean && hexo g && hexo algolia即可

  • (注意)每当有新文章发布，都需输入hexo g与hexo algolia指令

主题链接替换思路一(弃用)

由于jsdelivr的加速链接中有便捷的组合链接(即/cdn.jsdelivr.net/combine/npm_url1,npm_url2,..)，并且其只发送一次请求。以此来看，无论是修改js的请求代码还是直接添加多个js标签，都是发送同等数量的请求。

  • (step.1修改主题插件链接前缀)

    由于拆分了fancybox，导致图片无法点击放大或将一系列图片整成一个合集。此处还没弄清楚，也懒得弄了。具体是由于fancybox拆分后变成了jquery，fancybox和justifiedGallery三块。fancybox和justifiedGallery应该是jquery.fancybox和jquery.justifiedGallery，两者是jquery的子方法，需要先调用jquery插件这个先决条件，否者会报错。经多次测试后，仍然有问题，因此直接弃用了，拆分是为了缓解报错。

    unpkg的加速链接格式：unpkg.com/:package@:version/:file

    修改文件：hexo根目录./themes/shoka/_config.yml文件

    修改文件页面滑到最后，将原来的vendors元素所有内容注释(这里的comment与valine表示本地路径)
