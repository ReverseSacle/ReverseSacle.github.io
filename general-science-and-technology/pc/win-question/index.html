关闭win10热门搜索

win + x接着选择Power Shell(管理员)，接着输入指令

reg add HKCU\Software\Policies\Microsoft\Windows\explorer /v DisableSearchBoxSuggestions /t reg_dword /d 1 /f

会导致搜索框网络无法使用

卸载windows纸牌游戏与xbox与新闻和天气

win + x接着选择Power Shell，接着输入指令

新闻和天气 - Get-AppxPackage *bing* | Remove-AppxPackage

纸牌游戏 - Get-AppxPackage *solit* | Remove-AppxPackage

xbox - Get-AppxPackage *xbox* | Remove-AppxPackage

鼠标连接时自动关闭触摸板

win + i快捷键打开设置，输入连接鼠标时关闭触摸板，取消勾选连接鼠标时让触摸板保持打开状态

阻止流量绕过代理的加密隧道

Windows PowerShell下，输入指令

netsh interface teredo set state disabled

bing搜索取消搜索历史

进入必应主页面，点击顶栏右边的三条横 → 搜索历史记录 → 将在此显示最新搜索取消勾选 → 点击清除搜索历史记录

进入必应主页面，点击设置 → 更多 → 取消搜索建议处的键入时查看搜索建议 → 点击最后的保存

win桌面切换快捷键

Ctrl + WIN + → - 切换到下一个桌面

Ctrl + WIN + ← - 切换到上一个桌面

解决Microsoft Compatibility Telemetry的CPU占用率高

Win + R快捷键 → 输入taskschd.msc → 任务计划程序库 → Microsoft → Windows → Application Experience → 点击右边的Microsoft Compatibility Appraiser → 在下拉菜单中，单击禁用以禁用Microsoft Compatibility Telemetry服务

关闭谷歌浏览器自动更新

Win + R快捷键 → 输入services.msc → 找到Google Update(Google 更新服务) → 右键点击属性，设置为禁用

桌面找到Google Chrome的快捷标志 → 右键属性，在目标的最后添加--disable-background-networking

Win + R快捷键 → 输入taskschd.msc → 任务计划程序 → 右边找到带有Google的所有计划并设置为禁用

桌面找到Google Chrome的快捷标志 → 打开文件所在位置 → 返回到Google目录 → 进入Update目录 → 删除所有内容 → 返回到Google目录 → 右键Update目录，点击属性 → 安全 → SYSTEM → 底栏点击 拒绝 → 确认

Microsoft Office 2019 官方镜像下载地址合集

→专业增强版← | →专业版← | →家庭学生版← | →家庭企业版←

AMD笔记本CPU改善卡顿

windows桌面右小角的电池 → 右键选电池选项 → 更改计划设置 → 更改高级电源设置 →

PCI Express → 链接状态电源管理 → 接通电源选择 关闭

解析：将在pcie通道上的显卡的功率尽可能最大化，提高占用率和功率，以达到更稳定的帧数和游戏体验

AMD power slider → Overlay → 接通电源选择 最佳性能

解析：一个AMD的CPU的运行电源计划

AMD 图形能力设置 → AMD Powerplay设置 → 接通电源选择 优化性能

解析：尽可能使显卡满血和保持功率稳定的选项

IE浏览器的取消网页导航错误

打开Internet Explorer浏览器 → 点击右上角齿轮，选择Internet选项 → 点击高级选项 → 将检查发行商的证书是否已吊销与检查服务器证书是否已吊销的勾去掉 → 确认后重启电脑

台服和日服游戏乱码问题

可选方法一(换区)

进入windows语言设置中相关设置里面，进入管理页面，点击更改系统区域设置，设置为日语即可。

可选方法二(LocaleEmulator软件换区)

→Locale Emulator(密码:6r22)

下载LocaleEmulator软件后，打开LEGUI.exe，配置位置设定为日语，之后在游戏主程序右键中的LocaleEmulator中选Run in japanese。

台服/港服游戏中输入法无法输入繁体字(显示为?号)

通过windows自带的简体输入法的快捷键Ctrl + Shift + F，可在游戏中来回切换简体/繁体输入。

打开C盘根目录，可看到安装包

shift五次后弹出提示

Ax200网卡下载速度偏慢的处理方法

win + x快捷键 → 设备管理器中 → 网络适配器 → intel(R) Wi-Fi 6 AX200 160MHz → 右键选择属性 → 高级中选择Preferred Band → 值选择Prefer 5GHz band

笔记本蓝牙图标缺失及蓝牙驱动感叹号

原因是静电造成的，需关闭笔记本电源，将排线全部拔下，长按开机电源键12s左右，之后再次开启。

限制 Windows Defender 后台扫描默认CPU适用率

win + R快捷键输入gpedit.msc，计算机配置 → 管理模板 → Windows组件 → Microsoft Defender防病毒 → 扫描 → 指定扫描期间CPU使用率最大百分比 → 选择已启用并设置指定百分比为10
