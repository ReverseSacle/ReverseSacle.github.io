# 关闭win10热门搜索

win + R快捷键输入cmd，接着输入指令

reg add HKCU\Software\Policies\Microsoft\Windows\explorer /v DisableSearchBoxSuggestions /t reg_dword /d 1 /f

会导致搜索框网络无法使用

# 鼠标连接时自动关闭触摸板

win + i快捷键打开设置，输入连接鼠标时关闭触摸板，取消勾选连接鼠标时让触摸板保持打开状态

# 阻止流量绕过代理的加密隧道

Windows PowerShell下，输入指令

netsh interface teredo set state disabled

# bing搜索取消搜索历史

进入必应主页面，点击顶栏右边的三条横 → 搜索历史记录 → 将在此显示最新搜索取消勾选 → 点击清除搜索历史记录

进入必应主页面，点击设置 → 更多 → 取消搜索建议处的键入时查看搜索建议 → 点击最后的保存

# win桌面切换快捷键

Ctrl + WIN + → - 切换到下一个桌面

Ctrl + WIN + ← - 切换到上一个桌面

# 解决Microsoft Compatibility Telemetry的CPU占用率高

Win + R快捷键 → 输入taskschd.msc → 任务计划程序库 → Microsoft → Windows → Application Experience → 点击右边的Microsoft Compatibility Appraiser → 在下拉菜单中，单击禁用以禁用Microsoft Compatibility Telemetry服务

# 关闭谷歌浏览器自动更新

Win + R快捷键 → 输入services.msc → 找到Google Update(Google 更新服务) → 右键点击属性，设置为禁用

桌面找到Google Chrome的快捷标志 → 右键属性，在目标的最后添加--disable-background-networking

Win + R快捷键 → 输入taskschd.msc → 任务计划程序 → 右边找到带有Google的所有计划并设置为禁用

桌面找到Google Chrome的快捷标志 → 打开文件所在位置 → 返回到Google目录 → 进入Update目录 → 删除所有内容 → 返回到Google目录 → 右键Update目录，点击属性 → 安全 → SYSTEM → 底栏点击 拒绝 → 确认
