# 基础厨具及调味料

厨具：带柄的双耳炒锅，锅铲，煤气灶，菜刀，菜刀板

调味料：盐，味精，葱，姜，胡椒粉，耗油

# 切菜基础

一般左手用来稳住食材，右手执刀来切菜。

左手稳住食材时不要伸直手指，容易切到手，左手全部手指的手指中部与手指头部为一体，尽量向内部靠近。

右手执刀切时菜，微微向左偏斜，斜着切菜。

# 炒菜基础

# 关于炒锅的位置摆放

由于炒菜时，并不只是只炒一道菜，为此，炒完一道菜时还需要用水洗过，洗碗锅后需握住两端的握柄来将水倒掉。为此，倘若以中间成对称摆放，那两边的握柄都会非常烫。为此尽量将锅的左握柄往煤气灶中心方向靠，这样就只有右握柄非常烫了，这样就可以用锅铲钩住，然后提起，再通过左握柄提高锅的高度来将水倒掉。

# 关于加食用油量的计算

以炒锅中心的底部，即碰到炒锅锅底为圆点，延外一线向外展开到炒锅锅顶边缘为圆半径(由于炒锅并非平面，这里的圆是弧形圆)，将炒锅的半径分为三段，用大拇指与食指来比对物理空间上的距离即可，加食用油量要小于13\frac{ 1 }{ 3 }31​。

推荐将买回来的食用油用一个杯子类的容器盛着。

一般炒菜所加食用油量：用锅铲往盛食用油的杯子里铲油，铲两次

炒青菜所加食用油量：用锅铲往盛食用油的杯子里铲油，铲一次

# 关于如何握锅铲

想象一下你如何握筷子，就如何握锅铲。

# 关于如何控火

忙不过来时可以将火调小，例如要往锅中加东西的时候。

# 关于加盐量

一开加食用油时会加少量盐，之后就是菜熟了时加盐调味，接着就是尝味道时盐不够就加盐。

# 关于加味精量

一般十几粒就行了。

# 关于炒完菜之后的问题

炒完菜后，手上都会多多少少沾点油，为此每次炒完菜后最好用洗洁精洗手。

# 炒菜基本步骤

(注意) 这里的水指自来水，当然你也可以用矿泉水。

  1. 首先将炒锅洗干净，开煤气点火，待炒锅内的水蒸发后便可继续。 - 这一步是为了防止之后加食用油时油溅出。

  2. 接着，用铲来铲食用油并从炒锅侧边加入到锅中(倘若食用油是桶装的就直接从锅侧边倒)。

  3. 接着往食用油中加少量盐(凭感觉就是盐袋靠在铲上，倾斜稍微用力抖两次的量，用勺子来比喻就是铲一勺，用刀片沿着勺边缘划平后剩余的盐量)。 - 这一步是为了防止食用油加热溅出。

  4. 待炒锅内可见食用油上方有微微蒸汽时便可下菜。下菜时，不要从中间下，要从炒锅侧边下，要么使菜自行滑入食用油中或用铲从侧边铲入油中 - 这样做是为了防止加菜时菜中的水分让油溅出。

  5. 加完菜后便是用铲炒菜，用铲将菜铺开，铲菜时不用一直铲，铲菜时是从炒锅锅把手柄的这一边的侧边用铲铲到炒锅锅把手柄的这一边，这样就翻面了。

  6. 待菜看起来熟了时(菜熟没熟看颜色以及菜的外表软硬程度)，将火调小，将所有菜铲一边，再将多余的油用铲铲出到容器中(铲出的可以给下次用)。

  7. 接着铲一会菜(一会就一两分钟)，下香料(味精(十几粒就行了，就相当于在锅铲上轻轻的抖一两下的量)、葱、大蒜、酱油、辣椒等)，也下盐(加盐量尽量比步骤3中的加盐量少点)。

  8. 接着铲一会菜，用干净的容器盛水，将水加入到锅中，继续接着铲一会菜。

  9. 接着用铲铲出一点点菜水，用嘴常常盐加的够不够，不够就凭感觉加点盐。

  10. 加完盐继续铲菜，铲三四分钟后，就停火将菜铲出，菜水也要铲出。

  11. 这里已经停火了，之后就是洗锅，加适量水，将边缘的菜渣铲入到水中，左手拿锅左端握柄，右手用锅铲铲头与铲中间的长柄的衔接处的铲头边缘扣住锅右端握柄，用力提起来后，晃一晃，再将水倒入水槽中。

  12. 至此，炒菜的完整步骤就概况完了。

# 煎鸡蛋-案例

  1. (默认已经切好菜了)准备一个盛菜用的碗和一个加水用的碗，洗锅，开火，待锅中水分蒸发，加食用油，加盐。
  2. 用尖端部分轻轻敲碎鸡蛋但别敲开，只需有个裂开就行，双手拿住鸡蛋两端，右手大拇指按住鸡蛋裂口，双手移动到锅中油上方，尽量低点，但别碰到油(这是因为太高会让油溅出)，右手大拇指用一丁点力，掰开鸡蛋。
  3. 此时鸡蛋就在油中加热，看鸡蛋的外观状态，呈现扁平化并非液状即可翻面，待完全熟了之后，将煎好的这个鸡蛋铲到盛菜用的碗中，并继续下一个鸡蛋(与步骤2一样)。
  4. 待所需鸡蛋都煎好后，将盛菜碗中的鸡蛋都倒入到锅中，调小火，加(酱油，葱，盐，味精等)，再加水，调大火。
  5. 待锅水沸腾冒烟后，用铲铲非常少的菜水尝尝盐够不够，盐不够就加盐/盐太多就加少量水。
  6. 此时，用铲尽量将鸡蛋展开，一两分钟后就停火，就煎完鸡蛋，将鸡蛋盛入碗中就做好了。

# 炒青菜-案例

  1. (默认已经切好菜了，并且用一个盆子盛好了)准备一个盛菜用的碗和一个加水用的碗，洗锅，开火，待锅中水分蒸发，加食用油两铲，加盐。
  2. 将盛菜盆子中的菜倒入锅中(这里要注意，盛菜的盆中还有水，注意油溅出)，直接用铲将菜铺开。
  3. 待菜软了后，用铲铲菜翻面，待菜熟了差不多时，调小火，加(酱油，盐，味精等)，再加非常少量水，调大火。
  4. 继续铲菜，待菜熟了差不多时，用铲铲非常少的菜水尝尝盐够不够，盐不够就加盐/盐太多就加少量水。
  5. 继续铲菜，一两分钟后，停火，将菜铲到盛菜用的碗中。

# 炒鸡肉-案例

  1. (默认已经切好菜了，并且用一个盆子盛好了)准备一个盛菜用的碗和一个加水用的碗，洗锅，开火，待锅中水分蒸发，加食用油尽量靠近锅13\frac{ 1 }{ 3 }31​，加盐。
  2. 将盛菜碗中的鸡肉从锅的边缘，用锅铲扒入到锅中，将肉用锅铲铺平在油中。
  3. 待肉不是切菜时的样子时，便翻面，依次往复，待肉颜色变为偏黄色时，调小火，将肉扒到锅的一边，将锅中的油全部取出到盛油用的容器中。
  4. 将肉扒到锅中心，加水，加到水能铺平肉底部并微微溢出，将火调大，加(酱油，盐，味精，辣椒等)。
  5. 继续用锅铲炒菜，一两分钟后，加葱和姜，继续炒一两分钟，停火，将菜铲到盛菜用的碗中。

# 排骨玉米汤-案例

厨具：煲汤用的锅，煤气灶，一个盛菜用的盆子，长柄勺

材料：排骨、玉米、红枣

香料：盐、味精等

(默认排骨和玉米已切好)，玉米是切成几段的。

锅装满水，开火，待水沸腾后，将排骨倒入锅中，一两分钟中后，用勺将排骨全部捞出。调最小火，将锅中的水全部倒出，再装满水，调最大火，再将排骨倒入，将玉米倒入，边煮便将水面上的漂浮物用勺捞出，待捞的差不多时，加入香料，可以现在放红枣或者煮的差不多时再放，否则就会煮的稀烂，最后盖上锅盖，煮一个多小时就差不多了。

# 非寻常菜谱

# 半煎半煮豆腐鱼肉-案例

(前置条件)将鱼切成一块一块长方形的鱼肉并用盛器装载，分散加入少量盐，放入冰箱冷藏，半天即可，超多少也没关系。

准备制作好的鱼肉和接近正方形的小豆腐，煤气灶开火，加食用油到比较接近锅13\frac{ 1 }{ 3 }31​处，但不能达到该值。由于油量太多了，不应该一次性将鱼肉倒入，用锅铲一块一块的放入锅中。

等待锅中鱼肉可见面的肉块颜色变为偏黄色就翻面，翻面完再重复一次，之后将火调最小，将鱼肉捞出放到盛菜的盆中。

往盆中加入豆腐，再分散撒入各种香料(盐，味精，辣椒，葱，姜等)，之后将盆放入煮锅中。倘若有电磁炉可用电磁炉调1600煮20分钟上下即可，或者煮锅是可以用煤气灶烧的就用煤气灶烧10分钟左右即可。

注意关火。
