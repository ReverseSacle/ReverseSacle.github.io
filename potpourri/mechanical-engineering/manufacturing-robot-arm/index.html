# 恒正自动化的六轴机械手

# 基础知识

一轴~六轴分别用J1J_1J1​~J6J_6J6​来称呼，分别对应整个手的各个关键连接部位。

以常规的六轴机械手为例，单独控制键位有

  • J1J_1J1​对应整个手臂的左右旋转。

  • J2J_2J2​对应整个手臂的前后旋转。

  • J3J_3J3​对应前臂的前后旋转。

  • J4J_4J4​对应前臂的左右旋转。

  • J5J_5J5​对应手腕的前后旋转。

  • J6J_6J6​对应手腕的左右旋转。

固定原点为J1J_1J1​ ~J6J_6J6​全为0。

由于每个动作只能单独一个JiJ_iJi​动，因此，动作会有点卡顿。

编写应用场景类似于拼积木

  • 每个动作相当于一个积木
  • 当积木拼好后就被称为一个流程动作
  • 比如将锅从生产线上移动到另外一个位置的过程需要多个流程动作
  • 将每个流程动作的积木数量减少的过程相当于调优

# 基础界面

右侧是单轴运动控制键

  • - - 为顺时针方向旋转运动
  • + - 为逆时针方向旋转运动

底部是六轴联动控制键 - 分别为前/后/左/右/升/降。

界面中有

  • 打开运动程序

  • 保存运动程序 - 即保存现在所编写的运动轨迹

  • 编写运动程序

# 待确认厂家的五轴机械手配夹爪

# 基础详情

相当于，六轴机械手，但J6J_6J6​变成夹爪了。

首先是启动，每一个机械手都有对应的电箱，电箱后面有电源开关，注意当手动关闭电箱电源时需等待20秒(待继电器完全断电后)再开启电源。

# 基本界面

开机后需登录管理员用户，账号密码都为admin。

机械手是与通过总线信号来与其他生产工具连接的，是非联网的，但电箱内部有网口。

该产家的控制软件可以编写程序过程，但是需要知道对应生产设备所给的信号，为此产家会提前将程序过程编写好，再给出一个基本程序来修改对应的点位的数据改动接口。
